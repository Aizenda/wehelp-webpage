彈跳視窗CSS技術心得

1. 0%, 100% { transform: scale(1, 1); }: 在動畫的開始（0%）和結束（100%）時，元素的尺寸比例保持不變。
2. 25% { transform: scale(0.9, 1.1); }: 在動畫進行到25%的時候，元素水平方向（x軸）縮小到90%，垂直方向（y軸）放大到110%。
3. 50% { transform: scale(1.1, 0.9); }: 在動畫進行到50%的時候，元素水平方向放大到110%，垂直方向縮小到90%。
4. 75% { transform: scale(0.95, 1.05); }: 在動畫進行到75%的時候，元素水平方向縮小到95%，垂直方向放大到105%。

上面這段css是使用transform這個屬性的過度動畫來造成css的屬性變化，並通過@keyframes (0%-100%)的方式來控制css的屬性變化，最後我在class選擇器的基礎上再搭配滑鼠事件的偽類(hover) ，用以偵測滑鼠是否懸停在固定區域，並在其中設置style(animation: jielly-effect 0.5s;)來達成視窗彈跳的效果，並控制動畫撥放時間，雖然看起來沒有甚麼問題，但當滑鼠快速滑過，並且停留時間快過我設定的動畫時間的話，會造成動畫撥放不完全 ，這部分可以用javascript的模擬滑鼠動作來改善，但我目前javascript還不熟練，等到熟練後我會將這項效果技巧加以改善。
